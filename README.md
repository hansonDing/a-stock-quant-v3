# A股多因子轮动 × 趋势周期策略 V3.0

基于 AkShare 真实财务数据 + 腾讯K线缓存的量化选股回测系统。

---

## 核心特性

- **宏观定仓位**：5类市场状态动态调整总仓位（15%-100%）
- **风格定因子**：价值/成长/过渡期三套因子权重映射
- **行业轮动**：动量+趋势+资金流+拥挤度四维评分，自动筛选Top行业
- **多因子选股**：12个因子合成打分，MAD去极值 → 市值中性化 → Z-Score标准化
- **拥挤度过滤**：行业成交额占比历史分位防过热，>90% 扣分
- **三层风控**：个股止损(10%回撤) + 行业止损(破MA20/5日暴跌) + 大盘熔断(强熊市强制降仓)
- **AkShare真实数据**：PE/PB/ROE/净利润增长等真实财务因子，替代纯技术模拟

## 项目结构

```
├── config.py                  # 全局配置（参数、枚举、行业列表）
├── backtest_engine.py         # 回测主引擎，串联所有模块
├── macro_engine.py            # 宏观周期：市场状态 + 风格判断
├── industry_rotator.py        # 行业轮动：评分 + 筛选 + 拥挤度 + 止损
├── factor_model.py            # 多因子选股：因子处理 + 动态权重 + 行业内选股
├── portfolio_manager.py       # 组合管理：权重优化 + 交易执行 + 三层风控
├── data_layer.py              # 数据层基类
├── tencent_data_source.py     # 腾讯K线数据源（免费，无需Token）
├── akshare_data_source.py     # AkShare财务数据源（真实PE/PB/ROE）
├── main.py                    # 模拟数据回测入口
├── real_backtest.py           # 腾讯真实数据回测（2022-2024）
├── quick_backtest.py          # 快速回测（2024H1，40只蓝筹）
├── akshare_backtest.py        # AkShare真实数据回测（2024H1）
├── stock_picker.py            # 📌 选股推荐：基于最新数据输出下个交易日组合
└── visualization.py           # 可视化工具
```

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行回测

```bash
# 模拟数据回测
python main.py

# 腾讯真实数据回测（2022-2024）
python real_backtest.py

# AkShare 真实财务数据回测（2024H1）
python akshare_backtest.py
```

### 3. 生成选股推荐（基于最新数据）

```bash
python stock_picker.py
```

输出示例：
- 宏观状态判断（市场状态 / 风格周期 / 建议仓位）
- 入选行业（10个，含得分、拥挤度、权重）
- 因子权重分配
- 推荐个股（50只，每行业5只，行业等权）
- 组合概要（总权重、现金比例、行业分布）

结果自动保存为 `stock_picks_YYYY-MM-DD.csv`。

## 策略逻辑

**自上而下看方向，自下而上选个股，层层风控兜底。**

| 层级 | 模块 | 核心逻辑 |
|------|------|----------|
| **宏观** | `MacroCycleEngine` | 上证指数 MA20/60/120 + RSI + 成交额趋势 → 5类市场状态 + 仓位上限 |
| **风格** | `MacroCycleEngine` | 价值/成长指数比值(VGRatio) 的 20/60日均线 → 价值/成长/过渡期 |
| **行业** | `IndustryRotator` | 动量30% + 趋势30% + 资金流20% - 拥挤度20% → Top10行业 |
| **选股** | `FactorModel` | 12因子合成打分，MAD去极值→市值中性化→Z-Score标准化 → 每行业Top5 |
| **组合** | `PortfolioManager` | 波动率调整 + 行业上限15% + 个股上限3% + 留5%现金 |
| **风控** | `PortfolioManager` | 个股回撤10%止损 + 行业破MA20止损 + 强熊市强制降仓 |
